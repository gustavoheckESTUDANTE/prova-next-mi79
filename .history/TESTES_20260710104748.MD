- execução da aplicação: npm run dev (Funcionando)

- acesso à página inicial: pagina da rota / (funcionando)

- visualização dos cards de cursos: npm install && npm run de (funcionando)

- acesso à página de detalhes: pagina da rota /[id do curso] (funcionando);

- acesso à página sobre a WEG pagina da rota /sobre (funcionando);

- funcionamento dos links de navegação;

- conferência dos arquivos enviados ao repositório.

- carregamento dos dados;

- exibição da listagem a partir dos dados fornecidos;

- filtro da categoria Cursos, quando houver categorias diferentes;

- carregamento dos dados correspondentes na página de detalhes, quando houver rota dinâmica.
